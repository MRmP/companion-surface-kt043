## iDisplay KT043

This surface plugin provides native support for the iDisplay KT043 5x3 LCD button keypad.

### Setup

1. Connect the KT043 via USB.
2. Companion should automatically detect it as a surface.
3. The device appears with a 5-column by 3-row button grid.

### Features

- 15 programmable LCD buttons (72x72 pixels each)
- Brightness control via the Companion surface settings
- Button press/release detection

### Notes

- Only one KT043 device can be connected at a time (SDK limitation).
- The vendor SDK DLL must be present in the plugin's `vendor/` directory.
- Supported on Windows (64-bit) and Linux (64-bit).
